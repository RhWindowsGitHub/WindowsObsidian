## 一、 Nvm 概述

> Nvm 全称（Node Version Manager） Node 版本管理器，可用来同时安装多个版本 node 及丝滑切换 👍

## 二、Nvm 安装

>进入 [Nvm 官方下载页面](https://github.com/coreybutler/nvm-windows/releases/tag/1.1.11) 下载即可：

>验证：打开 `cmd` 输入 `nvm -v` 查看版本信息：

### 3.1 搜索可用的 node 版本

### 3.2 安装指定版本的 node

> `nvm install <版本号>`

### 3.3 切换不同的 node 版本

> `nvm use <版本号>`
